## 1.office 2016应用程序

### 1.1启动与退出

==启动==：

1. 通过“开始”菜单中的快捷方式启动
2. 通过桌面快捷方式启动
3. 通过关联文档启动

==退出==：

1. ==单击==应用程序==标题栏==右侧的“关闭按钮”。
2. ==双击==应用程序的==控制菜单==图标。（标题栏最左侧）
3. ==单机==应用程序的==控制菜单==图标（或在标题栏上右击），在控制菜单中选择“==关闭==”
4. 在应用程序窗口的文档中按`alt+F4`组合键
5. 右击任务栏图标➡关闭所用窗口

==注意==

若只想关闭相应的文档窗口而不退出应用程序，可使用的方法是：

1. 在应用程序窗口中按`ctrl + W`组合键
2. 在`backstage`视图中单机`关闭`按钮

文档所做的修改进行保存

1. 单击==保存==按钮，可以保存当前文档，同时关闭该文档。
2. 单击==不保存== 按钮，将直接关闭文档，且不会对当前文档进行保存，即文档中所做的更改都会被放弃。
3. 单击==取消==按钮，将关闭该提示对话框并返回文档，用户可根据实际需要进行相应的操作

### 1.4 office 2016的文档操作

#### 1.新建文件

​	利用`文件`选项卡中的`新建`命令，或者按下`ctrl +N`组合键，都可以创建一个新文件。以word2016为例，当新建word文档时，用户可以选择创建==空白文档==或者基于==模板创建文档==。==模板：预设内容和格式的文件==。

#### 2.保存新建文件

​	通过==快速访问工具栏==中的`保存`按钮，或者按下`ctrl + S`或者`shift +F12`组合键，或使用`文件`选项卡中的`保存`命令，以上操作均弹出“另存为”对话框，用户可根据需要设置保存路径及保存类型。

#### 3. 保存已有文件

​	同保存新建文件的方法相同，但不会弹出“另存为”对话框。

#### 4. 另存为新文件

​	**方法为**：打开`文件`选项卡，单击`另存为`命令或`F12`。需要注意，执行另存为命令时，用户可根基需要设置新的文档保存位置、新的文件名、新的保存类型等。

#### 5.自动保存文件

​	是为了==防止突然停电、死机==等意外情况而设置的功能。应用默认==每隔10分钟自动保存一次文档==。

​	自定义保存方法：打开`文件`命令，打开`word选项`对话框，在其`保存`选项卡中设置，默认`自动恢复信息时间间隔`为10分钟。

#### 6. 打开已有文件

​	打开已有文件，是将该文档从==外存==调入==内存==。

​	方法：打开`文件`选项卡，切换到`backstage`视图，点击`打开`命令，或者按下`ctrl + O`或者`ctrl + F12`组合键，或者在`backstage`视图中单击`最近所有文件`命令。

#### 7. 多个文档之间的切换

1. `视图`选项卡中的`窗口`组中的`切换窗口`命令。
2. `ctrl + F6`,或`alt +F6`。

#### 8.关闭文件

打开`文件`选项卡，单击`关闭`命令。如果文件编辑后没有保存，则系统将首先提示是否保存，然后关闭文件。需要注意：`关闭`命令的作用是关闭文档，而不是关闭软件本身。快捷键为`ctrl + W`。

#### 9.帮助信息

按`F1`

## 2. word2016 基本操作

### 2.1 文件视图
视图方式：

- 页面视图
- 阅读视图
- web版式视图
- 大纲视图
- 草稿视图

==**视图切换快捷键**==：

页面视图：`ctrl + alt + P`;

大纲视图：`ctrl + alt + O`;

草稿视图：`ctrl + alt + N`;

#### 1. 页面视图 `ctrl + alt + p` 

屏幕布局与打印输出的结果完全一样，能用于编辑页眉和页脚、处理分栏和编辑图像等对象，word2016默认视图方式为页面视图，可以显示水平标尺和垂直标尺。

#### 2. 阅读视图 

以图书的分栏样式显示word文档，不可编辑，没有水平垂直标尺

#### 3.web视图

可创建能显示在屏幕上的web页或文档，模拟该文档在web浏览器上浏览的效果，**仅能显示水平标尺**。

#### 4. 大纲视图`ctrl + alt + O`

主要用于设置word文档和显示标题的层级结构，可方便折叠、展开各层级文档内容，适用于长文档阅读，大纲视图不显示页边距、页眉和页脚、图片和背景。

#### 5. 草稿试图`ctrl + alt +N`

只显示文本格式，当文档满一页，出现一条虚线，称为分页符（<mark>自动分页符</mark>）,特点是占用计算机内存少，处理速度快。**仅能显示水平标尺**

所有视图都可以查看表格。

<mark>大纲视图和草稿视图不能显示背景和图片，所有视图都可以显示表格</mark>

### 2.2 文档的编辑

#### 1.创建文档

1. 创建空白文档，单击“`文件`”选项卡中的“`新建`”命令，快捷键`CTRL + N`

2. 根据模板创建文档。

#### 2. 文本输入

文档的输入窗口有<mark>一条闪烁的竖线</mark>,称为“<mark>插入点</mark>”,它指示<u>文本的输入位置</u>

文档的内容<u>包括汉字、英文字符、标点符号和特殊字符等</u>。

1. 选择合适的输入法

   `ctrl + space(空格)`：在中文输入法和英文输入法中切换；

   `ctrl + shift `：在各种输入法中切换。

2. 全角、半角字符输入

   <u>汉字没有全半角，全角和半角是两种不同字符</u>

   <u>一个全角字符相当于两个半角字符，可以使用快捷键`shift + space`在全半角之间切换</u>

3. 键盘上常见符号的输入

   键盘常见符号的输入包括标点符号和其他符号的输入

   中、英文标点符号的切换可以用以下两种方法来实现

   1. `ctrl + .`
   2. 单击输入法指示栏上的按钮进行切换

4. 特殊法号和难检字的输入方法
   1. 单击“`插入`”选项卡中的“`符号`”命令
   2. 使用软键盘

5. 插入点的定位

   插入点是在文档输入窗口中一条闪烁的竖线，用于知识文本的插入位置。word2016具有“即点即输”功能，即在空白页面的任意位置双击就可以输入文本

   <u>即点即输的功能只有在web版式视图和页面视图下才能使用</u>

   **快速定位快捷键**：

   1. `Home`:将光标从当前位置移至行首；
   2. `End`:将光标从当前位置移至行尾；
   3. `ctrl + Home`:将光标从当前位置移至文件开头；
   4. `ctrl + End`:将光标从当前位置移至文件结尾处；

6. 录入状态

   两种录入状态：“插入”和“改写”

   插入和改写状态切换方法：

   1. 按下键盘上的`Insert`键，可以在两种方式间切换；
   2. 单击状态栏上的改写/插入标记。

#### 3. 文档的修改与编辑

1. 选定文本

   1. 用鼠标选定文本：

      ​	**小块文本选定**：鼠标左键从起始位置拖动到终止位置

      ​	**大块文本选定**：起始位置单击一下，按住`shift`，单击终止位置，适用于连续大块跨页的文档，不连续文本使用`ctrl `选定

      ​	**选一行**：鼠标移至页左选定栏，指针变成向右的箭头，<u>单击</u>

      ​	**选定一句**：按住`ctrl `键同时，<u>单击句中的任意位置</u>

      ​	**选定一段**：鼠标移至左选定栏，<u>双击</u>，或者，<u>在段落中的任意位置<mark>三击</mark></u>

      ​	**选定整篇文档**：鼠标移至页左选定栏;快速三击，或标移至页左选定栏，按住`ctrl`键的同时单击鼠标，或使用`ctrl + A`组合键，这三种方法均可以选定整篇文档。

      ​	**矩形选择**：按住AIt键，在要选取的开始位置按下左键，拖动鼠标可以拉出一个矩形的选择区域。

   2. 用键盘选定文本：
   
      ​	`shift + ←（→）方向键`：分别向左(右)扩展选定一个字符；
   
      ​	`shift+ ↑（↓）方向键`：分别由插入点处向上(下)扩展选定一行；
   
      ​	`shift + Home`：选定播入点到行首；
   
      ​	`shift + End`:选定插入到行尾；
   
      ​	`ctrl + shift + Home`:从当前位置扩展选定到文档开头；
   
      ​	`ctrl + shift + End`:从当前位置扩展选定到文档结尾；
      
      
   
   3. 通过样式选择文本：

​			对文档应用样式后，可快速选定具有相同样式的所有文本，方法为:在“开始”选项卡的“样式“组中，右键单击某样式，在弹出的快捷菜单中选择“选择所有*个实例”命令(其中“`*`”表示当前文档中应用该样式的实例个数)。

2. 撤消文本的选定

   单击文本内的任意位置，即可以撤销文本的选定。

3. 删除文本

   按Backspace 键，向前删除光标前的字符;按Delete 键，向后删除光标后的字符。如果要删除大块文本，可采用如下方法:

   1. 选定文本后，按 Delete 键删除；
   2. 选定文本后，单击“`开始`”选项卡上的“`剪切`”命令,或单击右键,从快捷菜单中选择“`剪切`”命令，还可以使用`ctr1+X`组合键。

4. 移动文本
   1. 使用鼠标拖放移动文本
      1. 选定要移动的文本;
      2. 鼠标指针指向选定的文本，按住鼠标左键，鼠标指针尾部出现虚线方框，指针前出现一条竖直虚线；
      3. 拖动鼠标到目标位置，即虚线指向的位置，松开鼠标左键即可。
   2. 使用剪贴板移动文本
      1. 选定要移动的文本；
      2. 将选定的文木移动到剪贴板上(使用`ctrl + X`组合键);
      3. 将鼠标指针定位到目标位置，从剪贴板复制文本到目标位置(使用`ctrl + V`组合键)
   
5. 复制文本:常用的复制文本的方法:

   1. 用鼠标拖放复制文本
      1. 选定要复制的文本;
      2. 鼠标指针指向选定文本，按住`ctrl`键的同时，按住鼠标左键，鼠标指针尾部出现虚线方框和一个“+”号，针前出现一条竖直虚线;
      3. 拖动鼠标到目标位置，松开鼠标左键即可。
   2. 使用剪贴板复制文本
      1. 选定要复制的文本;
      2. 将选定的文本复制到剪贴板上(使用`Ctrl+C`组合键);
      3. 将鼠标指针定位到目标位置，从剪贴板复制文本到目标位置(使用`Ctrl+V `组合键)。

说明:<u>Word2016中最多可同时保存24项用户近期剪切或复制的内容。右键拖动文本，不仅可以移动文本，也可复制文本。</u>	

粘贴选项:保留源格式、合并格式、只保留文本。
选择性粘贴:word 文档对象、带格式文本、无格式文本、图片、HTML格式、无格式的unicode 文本。

### 2.4 文档校对

Word2016提供拼写和语法检查功能，能进行中英文的拼写和语法检查，可大大减少输入错误率，使单词、词语和语法的准确率得到提高。读者需要注意，Word给出的仅仅是建议，在很多情况下 Word 会出现误判现象。

#### 1.拼写和语法检查

 切换到“`审阅`”选项卡，在“`校对`”组中单击“`拼写和语法`”按钮，或者快捷键`F7`都可以进行打开拼写和语法检查功能。

<u>Word2016用<mark>红色波浪线标出拼写的错误</mark>，<mark>用蓝色波浪线标出语法的错误</mark></u>。这些标出的测浪线不影响文档的打印，属于非打印字符。当然，有些特殊用法或写法，例如人名、地名等也会被 Word 误认为是错误的，可以不用理会。

#### 2.自动更正

为提高输入和拼写检查效率，Word提供“自动更正”功能，以将<u>字符、文本或图形替换成特定的字符、词组或图形</u>。方法为:单击“`文件`”选项卡的“`选项`”命令，在打开的对话框中切换到“`校对`”选项卡，然后单击“`自动更正选项`”按钮。

#### 3.数字统计

切换到“`审阅`”选项卡，在“`校对`”组中单击“`字数统计`”按钮，弹出“`字数统计`”对话框，对话框中显示了当前文档的<u>页数、字数、段落数、行数字符数(不计空格、计空格)、非中文单词.中文字符和朝鲜语单词</u>等信息。也可以对文档中任意选定部分内容进行字数统计。(可选文本框、脚注和尾注)

#### 4.撤销与恢复

**撤销**:作用是取消最近一次的操作，用户可以连续撤销。方法为:单击快速访问工具栏上的“`撤消`”按钮，或按下`Ctrl+Z `组合键。

**恢复**:是对撤销操作的反操作，快捷键是`Ctrl+Y`

撤销是后退，恢复是前进，先有撤销才有恢复。

## 3.文档格式化与排版

### 3.1 设置字符格式

#### 1.设置字体

**方法为**:<u>选中文本</u>后，在<u>悬浮工具栏中设置字体</u>，或者在“`开始`”选项卡的“`字体`”组中设置。在“`开始`”选项卡的“`字体`”组<u>右下角单击对话框启动器按钮</u>，可打开“字体”对话框做详细设置。字体设置中的<u>加粗</u>、<u>斜体</u>、<u>下划线</u>对应的快捷键分别是`Ctrl+B`、`Ct1+I`、`Ctrl+U`。

**注意**：号越大，磅值越大字越大;号越大，字越小。

#### 2.字符间距

字符间距是指<u>字符之间的距离</u>，用户可根据需要灵活调整。

**方法为**:在“`字体`”对话框的“`高级`”选项卡中，进行“`缩放`”、“`间距`”、“`位置`”的设置。间距可设置<u>水平字符之间的距离，位置可设置字符高低</u>的设置，包括<mark>标准、提升、降低</mark>。

#### 3. 突出显示文本

Word2016提供对文本的突出显示功能，即用鲜明的颜色对文本做标记，从而使文本看上去像用荧光笔做了标记。

**方法是**:在“`开始`”选项卡的“`字体`”组中选择“`以不同颜色突出显示文本`命令，或者选中文本后在悬浮工具中设置。

#### 4.格式刷

格式刷是一种快速格式复制工具，能将某文本对象的格式复制到另一个对象上，从而避免重复设置格式的麻烦。

**方法为**:选中需要复制的格式所属文本，单击“`剪贴板`”组中的“`格式刷`”按钮，<u>按住鼠标左键，拖动鼠标选择需要设置相同格式的文本，完成后释放鼠标。</u>连续使用格式刷需**双击**“`格式刷`”按钮，当不再需要格式刷时，再次<u>单击“格式刷”或按Esc键即可退出格式复制状态</u>。

快捷键是 `Ctl+Shift+C/V`。

#### 5.清除字体格式

其作用是清除所选文本的所有字体格式，只保留纯文本。

**方法为**:点击“`开始`”选项卡中“`字体`”组的“`清除格式`”按钮。

快捷键是`Ctrl+Shif+Z`。

### 3.2 设置段落格式

**段落**:是指<u>文档中两次回车之间的所有字符</u>，包括<u>段后的回车符</u>。段落格式主要是指<u>段落中行距的大小、段落的缩进、换行、分页、对齐方式</u>等。

缩进度量单位:厘米、镑、字符。

**行距**:行与行之间的距离;有单倍、1.5倍、2倍、多倍行距、最小值、固定值。这是设定标准行距的倍数的行距;最小值和固定值用于设定固定的镑值作为行距。

**缩进**:可将选定段落的左、右边距缩进一定的量。

**左缩进**:是指整个段落左边界距离页面左侧（左页边距）的缩进量。

**右缩进**:是指整个段落右边界距离页面右侧（右页边距）的缩进量。

**首行缩进**:是指段落首行第1个字符的起始位置距离段落其他行左侧的缩进量。大多数文档的首行缩进量为两个字符。
**悬挂缩进**:是指段落中除首行以外的其他行距离页面左侧的缩进量。悬挂缩进方式一般用于一些较特殊的场合,如杂志、报刊等。

**特殊格式**:包括无、悬挂缩进、首行缩进。

**间距**:可以在段前、段后设置一定的空白间隙,通常以“行’或“镑”为单位。

**对齐方式**:可以设置段落或文本左对齐、居中对齐、右对齐、两端对齐、分散对齐等，默认为两端对齐。
